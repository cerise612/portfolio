Fetch is a website and mobile application that provides users with all the resources needed to meet their pets needs in addition to creating a community where pet owners can share their experiences and opinions.

After passing through the welcome page the user lands on the main page. The focal point is a map of pet friendly locations based on their geolocation. This can be changed using the search function at the top of the screen. buttons on the map allow the user to filter by location type. A collapsable navbar on the left side of the screen allows for additional navigation. Users can rate a location or save it as a favorite.

Users can use the service without signing up until they wish to engage with the community or save favorite locations.

The user dashboard provides the user a snapshot of their information including their favorite locations.

A minimum amount of information is required to sign up. If a user wishes to provide additional information to the community or to update existing information they can do so from the update screen.
